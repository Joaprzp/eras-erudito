# Banco de hechos verificados para nuevas Tarjetas de pregunta

Fecha de investigación: 19 de julio de 2026. Este documento propone contenido **original** para el mazo privado: no reproduce la redacción ni las selecciones de las tarjetas físicas transcritas. Cada bloque señala una fuente primaria u oficial para volver a comprobarlo antes de incorporarlo al mazo.

## Criterios de uso

- Las tarjetas evitan sucesos que puedan cambiar (cargos, rankings, cifras anuales). Las cifras de **Aproximación** especifican el período o la definición exacta.
- Para **En Común**, la solución debe admitir variantes razonables de redacción. El Anfitrión conserva la decisión final según las reglas de la Versión Familiar.
- Las cifras están pensadas para comparar cercanía, no para exigir una unidad distinta: el valor numérico es el que debe guardar la aplicación.
