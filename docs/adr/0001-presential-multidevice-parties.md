# Usar partidas presenciales multi-dispositivo con pantalla companion

El MVP será una experiencia presencial en la que los participantes se conectan desde sus teléfonos y una pantalla companion muestra el estado y las consignas compartidas. Elegimos este formato, en lugar de una pantalla única, para conservar la privacidad de las respuestas y poder medir con precisión la rapidez de cada equipo en la Versión Familiar.
