# Usar Bun, React, Convex y Cloudflare para el MVP

El MVP usará Bun como runtime y herramienta de proyecto; React 19 con TypeScript y Vite para el frontend; TanStack Router con rutas basadas en archivos; y Convex como backend y fuente autoritativa de datos en tiempo real. El frontend estático se desplegará con Cloudflare Workers Static Assets y Convex se desplegará por separado. La primera versión jugable se desplegará de manera privada para probar el companion y los teléfonos en condiciones reales. Usaremos Tailwind CSS v4 con tokens CSS/OKLCH, Base UI, Zustand solo para sesión local y preferencia de tema, y CVA, clsx y tailwind-merge para composición de estilos. Al implementar, se elegirán las últimas versiones estables que sean mutuamente compatibles y se validarán antes de instalarlas.
